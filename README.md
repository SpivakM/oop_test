## Лабораторна 5: Logistics System

### Запуск
- H2 (за замовчуванням): `./mvnw spring-boot:run`
- PostgreSQL: `SPRING_PROFILES_ACTIVE=postgres ./mvnw spring-boot:run`

### Змінні середовища для PostgreSQL
- `POSTGRES_URL` (за замовчуванням: `jdbc:postgresql://localhost:5432/logistics_db`)
- `POSTGRES_USER` (за замовчуванням: `postgres`)
- `POSTGRES_PASSWORD` (за замовчуванням: `postgres`)
